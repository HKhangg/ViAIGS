# ViAIGS: Vietnamese AI-Generated Social Media Text Detection

## Overview

ViAIGS is an experimental project for detecting Vietnamese social-media text written by humans versus text generated by large language models. The project focuses on common Vietnamese online platforms such as Facebook, VOZ, and YouTube, and evaluates several families of detection and analysis methods:

- Fine-tuned transformer models for human/machine text classification.
- Pre-trained AI text detectors.
- Statistical metrics based on probabilistic and linguistic features.
- Similarity analysis between machine-generated and human-written text.
- LLM-as-a-judge evaluation using Meaning, Naturalness, and Social Style criteria.
- A Streamlit demo that visualizes token-level attribution heatmaps to explain model predictions.

## Project Structure

```text
ViAIGS/
+-- datasets/                 # Experimental datasets
+-- notebooks/                # Preprocessing, statistics, and result analysis notebooks
+-- outputs/                  # Generated outputs and experiment artifacts
+-- src/
|   +-- app_token_explain.py  # Streamlit token attribution heatmap app
|   +-- finetune.py           # Fine-tune classification models
|   +-- generate_text.py      # Generate text with LLM generators
|   +-- metrics.py            # Compute evaluation metrics and statistics
|   +-- llm_judges/           # LLM judge evaluation pipeline
+-- requirements.txt
```

## Installation and Usage

Install the required dependencies:

```bash
pip install -r requirements.txt
```

Run the token attribution explanation app:

```bash
streamlit run src/app_token_explain.py
```

The Streamlit app accepts a JSON file containing a list of samples. Each sample can follow this structure:

```json
{
  "sample_id": 0,
  "text": "...",
  "source": "fb",
  "category": "TP",
  "predicted_label": 1,
  "true_label": 1,
  "tokens": [
    {
      "position": 0,
      "raw_token": "<s>",
      "clean_token": "<s>",
      "score": 0.0
    }
  ]
}
```

## Key Findings

- The ViAIGS dataset contains 24,593 samples split into train/dev/test sets, with a relatively balanced distribution across Facebook, VOZ, and YouTube.
- Among the paraphrasing strategies, Paraphrasing 3 provides a moderate similarity level rather than being overly close to human-written text.
- GPT obtains the highest average quality score under the LLM judge setting, especially on the Social Style criterion.
- Fine-tuned models clearly outperform pre-trained detectors and statistical metrics on the full dataset distribution.
- Meta-Llama-3-8B is the strongest model across most evaluation settings, including both in-distribution and out-of-distribution tests.
- Cross-source results indicate that the fine-tuned models generalize well across Facebook, YouTube, and VOZ.
